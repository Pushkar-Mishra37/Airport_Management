# Flight_Management
# ✈️ Flight Management System (Python)

A simple command-line based Flight Management System built using Python and `pickle` for data storage. This project allows users to manage flight records and perform operations like adding, updating, deleting, and booking seats.

---

## 📌 Features

- Add new flights  
- View all flights  
- Search flight by flight number  
- Update flight details  
- Delete flight  
- Book seats on a flight  
- Persistent storage using `pickle`  

---

## 🛠️ Technologies Used

- Python 3
- Pickle (for file handling and data storage)
